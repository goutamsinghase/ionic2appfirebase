### With the CLI:

To run it, cd into `ionic2appfirebase` and run:

```bash
$ sudo npm install -g ionic cordova
$ npm install
$ ionic serve
```


### Features 

1. Login 
2. Logout
3. Post your thoughts like twitter
4. View posts all from all users

### Node and Ionic version

```bash
$ npm -v  --> 5.0.4
$ node -v  --> 6.10.0
```
